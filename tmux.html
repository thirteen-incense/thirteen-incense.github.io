Layouts 布局

层级 描述
server 服务
session 会话
window 窗口
pane 窗格

Window 窗口

命令 描述
c 新建窗口
& 关闭窗口
l 切换窗口
n 切换到下一个窗口
p 切换到上一个窗口
w 窗口的菜单列表

Pane 窗格

命令 描述
% 水平分屏
" 垂直分屏
x 关闭窗格
; 切换窗格
o 顺时针切换窗格
C-o 逆时针转换窗格
M-o 顺时针转换窗格

配置1 , 2

新建 ~/.tmux.conf 文件，并写入： 

 1: # Set new default prefix
 2: # 修改主键
 3: unbind C-b
 4: set-option -g prefix C-j
 5: 
 6: # Mouse
 7: # 激活鼠标模式
 8: set-option -g -q mouse on
 9: 
10: # Easy split pane commands
11: # 修改分屏按键
12: bind h split-window -h
13: bind v split-window -v
14: unbind '"'
15: unbind %
